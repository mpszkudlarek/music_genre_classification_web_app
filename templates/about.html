Project Insights

Deep Learning Explorations

In the deep learning section, the main focus was:

  • Preprocessing: With the Librosa library, extracting features like MFCCs, Chroma, and RMS to set the stage for sophisticated analysis.
  • Model Training: Those features were channeled into a Convolutional Neural Network (CNN), training it to discern and categorize music genres with precision.

Web Programming

Flask is minimalist framework in Python that empowers to weave together the threads of web development and machine learning. In this project, Flask serves as the backbone for the deep learning models to life on the web.

  • Deployment: Combining Flask with deep learning model allowed to craft the music genre recognition on the home page

GitHub Integration

GitHub is a version control and collaborative development of this project. It hosts the source code and provides tools for code management.

  • Version Control: GitHub is used to track changes, manage branches, and maintain the history of the project's development.
  • Repository: check out the project repository on my GitHub profile.
