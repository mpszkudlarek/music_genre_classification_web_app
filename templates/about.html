Project Insights

Deep Learning Explorations

In the deep learning section, the main focus was:

  • Preprocessing with the Librosa library, extracting features like MFCCs to set the stage for a sophisticated analysis.
  • Channeling those features into a Convolutional Neural Network (CNN), training it to recognize and categorize music genres.

Web Programming

Flask is a minimalist framework in Python that empowers to weave together the threads of web development and machine learning.

In this project, Flask serves as the backbone for the deep learning models to life on the web.

Combining Flask with deep learning model allowed to craft the music genre recognition on the home page.
